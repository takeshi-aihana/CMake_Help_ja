.. versionchanged:: 3.27

  バージョン 3.5 より古い CMake との互換性は廃止になりました。
  CMake バージョン3.27 以上では、最低バージョンとしてバージョン 3.5 を指定しない :command:`cmake_minimum_required(VERSION)` や :command:`cmake_policy(VERSION)` の呼び出しは「推奨しない」旨の警告が表示されます。

.. versionchanged:: 3.19

  バージョン 2.8.12 より古い CMake との互換性は廃止になりました。
  CMake バージョン 3.19 以上では、最低バージョンとしてバージョン 2.8.12 を指定しない :command:`cmake_minimum_required(VERSION)` や :command:`cmake_policy(VERSION)` の呼び出しは「推奨しない」旨の警告が表示されます。
